Parts of a Computer

Here, you can find the main parts of a computer.

Name Function Picture
CPU Does all of the calculations for the computer. Without the CPU, a computer can't operate.
Hard Drive Stores all of the data that isn't needed immediately. Very slow but can hold enormous amounts of info.
RAM A small memory that the CPU can easily get data from. Basically the opposite of a hard drive: fast but limited.
Motherboard A large piece of fibreglass and copper that holds and connects all of the computer's components together.
Network Card Allows the computer to connect to a wired network
Input and Output Devices Allows the user to interact with the computer and recieve data from it.
